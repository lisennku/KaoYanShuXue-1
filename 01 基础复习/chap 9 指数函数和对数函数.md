# 1. 基础知识

## 1.1 指数函数

> 形如$f(x) = a^x　a > 0$的函数被称为指数函数

指数函数法则：

- $a^0 = 1$
- $a^1 = a$
- $a^xa^y = a^{x+y}$
- $\frac {a^x} {a^y} = a^{x-y}$
- $(a^x)^y = a^{xy}$

## 1.2 对数函数

> 形如$f(x) = log_a(x)　a > 0 且 a \ne 1, x > 0$的函数称为对数函数

对数函数法则：

- $log_a(1) = 0$
- $log_a(a) = 1$
- $log_a(xy) = log_a(a) + log_a(b)$
- $log_a(x/y) = log_a(x) - log_a(y)$
- $log_a(x^y) = ylog_a(x)$
- <font size = 4>$log_a(x) = \frac {log_c(x)} {log_c(a)}$</font>
